Speech Command Recognition 
این پروژه یک مدل تشخیص گفتار (Speech Command Recognition) است که با استفاده از دیتاست شامل نمونه‌های صوتی کلمات، آموزش دیده و می‌تواند هر فایل صوتی جدید را دریافت کرده و تشخیص دهد چه کلمه‌ای در آن گفته شده است.
پردازش و استخراج ویژگی از فایل‌های صوتی (MFCC, Spectrogram, Mel-Spectrogram)
آموزش مدل برای تشخیص کلمات از روی صدا
اجرای مدل روی هر فایل صوتی جدید برای پیش‌بینی کلمه گفته‌شده
تولید خروجی به‌صورت:
پیش‌بینی کلمه
درصد اطمینان مدل
احتمال هر کلاس
تکنولوژی‌ها و ابزارها

Python
Librosa برای پردازش صوت
NumPy, Pandas
TensorFlow/Keras
Matplotlib, Seaborn برای رسم نمودارها
Jupyter Notebook
